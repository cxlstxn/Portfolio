The project you are using right now!


This project I am unbelievably proud of, I felt as if I needed a portfolio that really just represents me and a boring old webpage from 2002 wasnt going to work. So I got to work creating the html and css for a theme that I have always been fond of, a Retro Terminal.


It is a simple portfolio that showcases my projects, skills, and experience in a fun and unique way. The terminal interface allows you to navigate through my work as if you were using a command line. Featuring custom commands along with some hidden terminal commands that work on the main page. It was really quite intuitive and fun to work on these outside of coding an actual shell.


Upon starting this project it really threw me back to my Unix Shell project, working with terminal logic and commands, though obviously quite different as its coded in JS. This project also helped me learn how to work along side web services like AWS and how to host on my own custom domain.


It was a really fun project to work on, and I hope you enjoy using it as much as I enjoyed creating it. Feel free to explore the project and please if you have any feedback at all dont hesitate to contact me.
